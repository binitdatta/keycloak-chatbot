What this app is all about

Natural language → JSON payload → Keycloak Admin REST API. No more hunting through documentation.

Design goal: AI-powered admin tool + SSO secured

Secure SSO Authentication

Full OIDC Authorization Code Flow with PKCE. No passwords stored — session managed via Keycloak tokens.

Claude-Powered Intent Engine

Anthropic Claude parses English instructions and extracts precise JSON payloads, mapping to the correct API endpoint.

LangGraph Agent Pipeline

Multi-step graph pipeline: Parse Intent → Execute API → Format Response. Stateful, observable, extensible.

Full Admin REST API Coverage

Users, clients, roles, groups, identity providers, client scopes, protocol mappers, realm settings — all in one chat.

Transparent Operations

Every operation shows the exact API payload and HTTP response. You always know what was sent to Keycloak.

Python 3.12 + FastAPI

Production-grade async backend with FastAPI, LangChain, LangGraph and Anthropic SDK. Easy to extend and deploy.

Full Keycloak management in chat

Just describe your intent. KeycloakAI understands context, handles edge cases, and confirms every operation with the live API response.

How KeycloakAI connects to the enterprise

Designed to mirror real enterprise patterns: identity, APIs, agent orchestration, and observability.

Identity & Access
  • Keycloak as IdP (OIDC PKCE)
  • PKCE public client — no secret stored
  • Session cookie — no JWTs on the client
AI Pipeline
  • Claude parses intent → JSON payload
  • LangGraph routes to correct API node
  • Deterministic branching — no hallucinated endpoints
Operations
  • Every API call shown verbatim
  • HTTP status + response body surfaced
  • FastAPI async backend — production-grade
Extensibility
  • Add new intents in agent.py
  • New API calls in keycloak_client.py
  • Docker-compose ready for deployment
