Why KeycloakAI

Built for security teams

A complete AI-powered admin interface backed by real Keycloak Admin REST API calls — not simulated.

Secure SSO Authentication

Full OpenID Connect Authorization Code Flow with PKCE. No passwords stored — your session is managed via Keycloak tokens.

Claude-Powered Intent Engine

Anthropic Claude parses your English instructions and extracts precise JSON payloads, automatically mapping to the correct API endpoint.

LangGraph Agent Pipeline

Multi-step graph-based pipeline: Parse Intent → Execute API → Format Response. Stateful, observable, and extensible.

Full Admin REST API Coverage

Users, clients, roles, groups, identity providers, client scopes, protocol mappers, realm settings — everything in one chat interface.

Transparent Operations

Every operation shows you the exact API payload and HTTP response. You always know precisely what was sent to Keycloak.

Python 3.12 + FastAPI

Production-grade async backend built with FastAPI, LangChain, LangGraph and Anthropic SDK. Easy to extend and deploy.
